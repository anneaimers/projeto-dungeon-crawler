# ⚔️ Dungeon Crawler

Seja bem-vindo ao Dungeon Crawler! Com a arma de sua preferência, seu objetivo é retomar o controle da masmorra. Monstros rastejantes e caçadores perigosos invadiram os corredores e espalharam espinhos por toda parte, bloqueando as passagens. 

Preciso de você para abrir o caminho até a câmara principal, onde o **Imperador Zurg** espera. Você vai ousar desafiá-lo?

---

## 🎯 Objetivo do Jogo

Abra caminho pelos corredores perigosos, destrua caixas, colete chaves e abra portas. Fique atento aos botões espalhados pelo mapa e descubra suas funcionalidades. 

> 💡 **Dica :** Lembre-se de priorizar os botões (`O`) para ativar efeitos específicos em cada andar!

---

## 🎮 Comandos de Controle

Utilize as teclas do seu teclado para movimentar o personagem e interagir com o cenário:

| Tecla | Ação |
| :---: | :--- |
| **W** | Mover para Cima |
| **A** | Mover para a Esquerda |
| **S** | Mover para Baixo |
| **D** | Mover para a Direita |
| **I** | Interagir (Coletar itens / Ativar botões) |
| **O** | Atacar / Destruir obstáculos |

---

## 🧩 Elementos e Símbolos do Mapa

Conheça o significado de cada símbolo que você encontrará pelos corredores da masmorra:

### 🤠 O Jogador
* `<` : Jogador olhando para a **esquerda**
* `^` : Jogador olhando para **cima**
* `>` : Jogador olhando para a **direita**
* `v` : Jogador olhando para **baixo**

### 🧱 Cenário e Obstáculos
* `*` : **Parede** – Bloqueia a passagem do jogador.
* `#` : **Espinho** – Perigo mortal! O jogador morre ao passar por cima.
* `k` : **Caixa** – Bloqueia o caminho, mas pode ser destruída com um **Ataque (`o`)**.
* `O` : **Botão** – Aciona efeitos específicos em cada andar.
* `D` : **Porta Fechada** – Bloqueia a passagem até ser aberta.
* `@` : **Chave** – Abre uma porta fechada específica ao **Interagir (`i`)**.
* `=` : **Porta Aberta** – Passagem livre para o jogador.
* `L` : **Escada** – Leva o jogador para a próxima fase.

### 👾 Inimigos
* `X` : Monstro Tipo 1 (Rastejante)
* `Y` : Monstro Tipo 2 (Caçador)
* `Z` : **Boss Final (Imperador Zurg)** 👑
