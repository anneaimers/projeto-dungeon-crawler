# projeto-dungeon-crawler

Com a arma de sua preferencia, retome o controle da masmorra. Monstros rastejantes e caçadores invadiram os corredores e distribuiram espinhos perigosos por toda a parte bloqueando passagens. Preciso de voce para abrir o caminho ate a camara principal, onde o Imperador Zurg espera. Vai ousar desafia-lo?

Destrua caixas, colete chaves, abra portas e descubra as funcionalidades dos botões (Dica: lembre-se de prioriza-los)
Comandos:
w - Cima
a - Esquerda
s - Baixo
d - Direita
i - Interagir
o - Atacar

Simbolos:
< - Jogador olhando para a esquerda
^ - Jogador olhando para cima
> - Jogador olhando para a direita
v - Jogador olhando para baixo
* - Parede: o jogador nao pode passar
# - Espinho: o jogador morre ao passar por cima
k - Caixa: o jogador nao pode passar, mas pode ser destruida com ataque
O - Botao: aciona efeitos especificos em cada andar
D - Porta fechada: o jogador nao pode passar
@ - Chave: abre uma porta fechada especifica ao interagir
= - Porta aberta: o jogador pode passar
L - Escada: leva o jogador para a proxima fase
X - Monstro Tipo 
Y - Monstro Tipo 2
Z - Boss Final
